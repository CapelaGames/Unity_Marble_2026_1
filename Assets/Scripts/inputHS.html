High Score Name Input — InputHS.cs

This script shows a name-entry panel on the win screen only when the player beats the high score. The player types their name and hits Submit to save it.

The Script — create Assets/Scripts/InputHS.cs

using UnityEngine;
using TMPro;

public class InputHS : MonoBehaviour
{
    public SaveToFile saveToFile;
    public GameManager gameManager;

    public GameObject nameInputPanel;   // the panel that appears when a new high score is set
    public TMP_InputField nameField;    // the text input field inside that panel

    float pendingTime;                  // stores the score until the player submits a name

    // Call this from GoalTrigger when the player wins
    public void CheckHighScore()
    {
        float timeRemaining = gameManager.timer;
        SaveData current = saveToFile.GetHighScore();

        if (timeRemaining > current.bestTime)
        {
            pendingTime = timeRemaining;
            nameInputPanel.SetActive(true);   // show name entry panel
        }
    }

    // Attach this to the Submit button's OnClick event
    public void Submit()
    {
        string playerName = nameField.text.Trim();

        if (playerName == "")
            playerName = "Player";   // fallback if they leave it blank

        saveToFile.NewScore(pendingTime, playerName);
        nameInputPanel.SetActive(false);
    }
}

Update GoalTrigger.cs

Replace the direct saveToFile.NewScore(...) call with inputHS.CheckHighScore():

public class GoalTrigger : MonoBehaviour
{
    public GameObject winGame;
    public GameManager gameManager;
    public InputHS inputHS;           // replace saveToFile with inputHS

    void OnTriggerEnter(Collider other)
    {
        PlayerController player = other.GetComponent<PlayerController>();
        if (player != null)
        {
            inputHS.CheckHighScore();    // shows name panel only if it's a new record
            winGame.SetActive(true);
            Time.timeScale = 0;
        }
    }
}
Note: Time.timeScale = 0 pauses the game but UI input fields still work fine — Unity's event system is not affected by timeScale.

Build the UI in Unity

1. Create the Name Input Panel
  1. In the Hierarchy, right-click the Canvas → UI → Panel. Name it NameInputPanel.
  2. Inside it, add:
    • UI → Text - TextMeshPro — label it "New High Score! Enter your name:"
    • UI → Input Field - TextMeshPro — name it NameField
    • UI → Button - TextMeshPro — label it Submit
  3. With NameInputPanel selected, uncheck the checkbox in the Inspector to disable it by default.
2. Add InputHS to the scene
  1. Select any persistent GameObject (e.g. your GameManager object).
  2. Click Add Component and add InputHS.
  3. Assign the fields in the Inspector:
    • Save To File → drag your SaveToFile GameObject
    • Game Manager → drag your GameManager GameObject
    • Name Input Panel → drag NameInputPanel
    • Name Field → drag the NameField Input Field
3. Wire the Submit button
  1. Select the Submit button in the Hierarchy.
  2. In the Inspector under On Click (), click the + button.
  3. Drag the GameObject that has InputHS into the object slot.
  4. In the function dropdown, choose InputHS → Submit().
4. Update GoalTrigger in the Inspector
  1. Select the GoalTrigger GameObject.
  2. Remove the old Save To File field (it's gone from the script now).
  3. Assign the Input HS field → drag the GameObject that has InputHS.
  4. Assign Game Manager as before.
