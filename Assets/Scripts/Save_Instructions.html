Save.cs — High Score Setup Instructions

The Save.cs script saves the player's best time (time remaining on the clock when winning) to a JSON file. Follow the steps below to wire it up in the Unity Editor.

Step 1 — Add Save to the Scene

  1. In the Hierarchy, create an empty GameObject: GameObject → Create Empty.
  2. Name it SaveManager.
  3. With SaveManager selected, click Add Component in the Inspector and add Save.
The script uses DontDestroyOnLoad, so it will persist across scenes automatically.

Step 2 — Update GoalTrigger.cs

Open GoalTrigger.cs and make the following changes:

Add a public field for GameManager:

public GameManager gameManager;

Inside OnTriggerEnter, before winGame.SetActive(true), add:

if (Save.instance != null && gameManager != null)
    Save.instance.TrySetHighScore(gameManager.timer);

The finished method should look like:

void OnTriggerEnter(Collider other)
{
    PlayerController player = other.GetComponent<PlayerController>();
    if (player != null)
    {
        if (Save.instance != null && gameManager != null)
            Save.instance.TrySetHighScore(gameManager.timer);

        winGame.SetActive(true);
        Time.timeScale = 0;
    }
}

Step 3 — Assign the GameManager Reference

  1. Select the GoalTrigger GameObject in the Hierarchy.
  2. In the Inspector you will now see a Game Manager field.
  3. Drag the GameObject that has your GameManager component into that field.

Step 4 — (Optional) Display the High Score

To read the saved high score anywhere in your code (e.g. on the main menu), call:

float best = Save.instance.GetHighScore();

Then display it with a TextMeshPro or UI Text element.

Where is the file saved?

The high score is written to Application.persistentDataPath/highscore.json.

On Windows this is typically:
C:\Users\<YourName>\AppData\LocalLow\<CompanyName>\<GameName>\highscore.json
